Web Development: Platforms, Languages, Frameworks, and CMS

  • Non-Java Languages
    • PHP: Symfony, Zend Framework, CodeIgniter, CakePHP (MVC), Drupal (CMS), Joomla (CMS), Concrete5 (CMS)
    • Ruby: Rails (MVC)
    • Python: Django (MVC), Zope, Plone (CMS)
    • Perl: Catalyst (MVC), Jifty, Template Toolkit, ...
    • ColdFusion
  • Java: Servlets, JSP, JSTL, JSF, Struts, Grails, Apache Wicket.
  • Microsoft
    • Active Server Pages, C#, .NET: Sharepoint (CMS)
  • Web "Content Management" Systems
    • CMS systems - Drupal, Joomla, HippoCMS, MS Sharepoint, Apache Lenya, ...
    • Blog software - Wordpress, TypePad, MovableType, ...
    • Wiki software - Media Wiki, XWiki, TWiki, ...
